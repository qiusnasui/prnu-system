# PRNU 设备指纹溯源系统

基于图像传感器 PRNU（Photo Response Non-Uniformity）噪声模式的设备指纹识别与图像来源鉴定系统。

## 功能特性

### 核心算法
- **PRNU 指纹提取**: 从参考图像集中提取设备特有的噪声模式
- **噪声残差计算**: 使用中值滤波/小波变换提取待测图像的 PRNU 噪声
- **相似度匹配**: 基于互相关和峰值能量比的综合相似度计算
- **判定结论**: 自动输出匹配分数和来源判定结论

### Web 界面
- 可视化上传参考图集和待测图像
- 一键执行比对分析
- 直观显示相似度分数（0-100）
- 判定结论和分析建议

## 使用方法

### 步骤 1: 上传参考图像
- 上传来自**同一设备**的多张图像（建议 5-10 张）
- 支持多选上传或拖拽上传
- 点击"开始上传参考图像"按钮

### 步骤 2: 上传待测图像
- 上传需要溯源的待检测图像
- 支持拖拽上传
- 点击"开始上传待测图像"按钮

### 步骤 3: 执行比对
- 点击"开始比对"按钮
- 系统自动计算相似度并输出结果

## 结果解读

| 分数范围 | 判定等级 | 说明 |
|---------|---------|------|
| 85-100 | 高度匹配 | 图像很可能来自同一设备 |
| 70-84 | 中度匹配 | 图像可能来自同一设备 |
| 50-69 | 低度匹配 | 图像来源不确定 |
| 30-49 | 不太匹配 | 图像可能来自不同设备 |
| 0-29 | 不匹配 | 图像很可能来自不同设备 |

## 算法原理

PRNU（Photo Response Non-Uniformity）是 CMOS/CCD 图像传感器的固有噪声模式，由制造过程中的微小差异导致。每个设备的 PRNU 模式都是独一无二的，可作为"设备指纹"使用。

### 处理流程
1. **噪声提取**: 使用中值滤波/小波变换分离图像的低频内容和高频噪声
2. **指纹构建**: 对多张参考图像的 PRNU 进行加权平均得到设备指纹
3. **相似度计算**:
   - 皮尔逊互相关系数
   - 峰值能量比（PER）
4. **综合判定**: 加权综合得分 + 置信度评估

## 注意事项

- 参考图像质量影响识别准确率，建议使用清晰、光线充足的图像
- 待测图像与参考图像分辨率差异过大会降低置信度
- JPEG 压缩会一定程度削弱 PRNU 信号
- 建议参考图像数量 ≥ 5 张以获得稳定指纹
